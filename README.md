# create-react-custom-app
Environment construction of React + Typescript + webpack + ESLint + Prettier

# How to use
```
npx create-react-custom-app <app_name> [-a <author>]
cd <app_name>
npm init
npm start
```
Open http://localhost:3000

# Build&Run
```
npm run build && npm start
```

# ESLint
```
npm run lint
npm run lint:fix
```

# Maintenance
```
npm run clean
npm run upgrade
```
